# Python

### Basic Concepts
- Exponentiations
```
>>> 2**5
32
>>> 9 ** (1/2)
3.0
```
- Quotient & Remainder
```
>>> 20 // 6   (floor division)
3
>>> 1.25 % 0.5
0.25
```
- Input
```
input("Enter something please: ")

float(input("Enter a number: ")) + float(input("Enter another number: "))
Enter a number: 40
Enter another number: 2
42.0

>>> foo = input("Enter a number: ")
Enter a number: 7
>>> print(foo)
7
```
- Strings
```
>>> print("spam" * 3)
spamspamspam

>>> 4 * '2'
'2222'
```
