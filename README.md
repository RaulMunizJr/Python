# Python

### Basic Concepts
- Exponentiations
```
>>> 2**5
32
>>> 9 ** (1/2)
3.0
```
- Quotient & Remainder
```
>>> 20 // 6   (floor division)
3
>>> 1.25 % 0.5
0.25
```
- Input
```
input("Enter something please: ")

float(input("Enter a number: ")) + float(input("Enter another number: "))
Enter a number: 40
Enter another number: 2
42.0

>>> foo = input("Enter a number: ")
Enter a number: 7
>>> print(foo)
7
```
- Strings
```
>>> print("spam" * 3)
spamspamspam

>>> 4 * '2'
'2222'
```
- If, Else
```
num = 7
if num == 5:
   print("Number is 5")
elif num == 11:
   print("Number is 11")
elif num == 7:
   print("Number is 7")
else:
   print("Number isn't 5, 11 or 7")
```
- logic
```
>>> 1 == 1 and 2 == 2
True
>>> 2 < 1 or 3 >  6
False
>>> not 1 == 1
False
```
